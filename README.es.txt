Changelogs te mantiene informado sobre las actualizaciones de tus aplicaciones Homey instaladas. La aplicación monitorea automáticamente todas tus aplicaciones en busca de cambios de versión y obtiene el registro de cambios correspondiente de la Homey App Store cada vez que se detecta una actualización.

Usa la tarjeta de activación para iniciar un flujo cuando una aplicación se actualiza, con acceso al nombre de la aplicación, la versión, la versión anterior y el texto completo del registro de cambios. La tarjeta de acción te permite consultar en cualquier momento el último registro de cambios de una aplicación instalada. Ya sea que quieras una notificación en la línea de tiempo, un mensaje push o cualquier otra automatización, Changelogs te proporciona los elementos necesarios.
